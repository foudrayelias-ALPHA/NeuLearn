# Neural Network Learning App

This repository starts with the product and implementation plan for a mobile-first educational app focused on teaching neural networks and machine learning through clear visuals, purposeful interactivity, and strong information design.

The app is designed for broad accessibility across experience levels, with content that can welcome a curious beginner while still supporting deeper exploration for more advanced learners.

## Direction

- Platform: iOS-first mobile app
- Recommended stack: SwiftUI
- Future expansion: Android via a later cross-platform content/model strategy
- Tone: educational, visually distinctive, and non-gimmicky
- Design rule: do not prioritize "fun" over understanding

## Files

- `docs/product-plan.md`: product vision, audience, learning structure, design principles, and feature definition
- `docs/swiftui-roadmap.md`: recommended technical architecture and phased build plan for a native iOS app

## Core Product Principles

- Start from what the learner already knows
- Make neural network behavior visible, not abstract
- Use interactivity when it improves understanding
- Keep advanced content dense and explorable rather than overly game-like
- Build a visual identity that feels crafted, human, and original
